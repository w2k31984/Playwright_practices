# 🎭 Playwright Practices
<hr>
<p>Este es un repositorio dedicado al aprendizaje y práctica del uso de Playwright , una herramienta potente y moderna para automatizar navegadores web (Chromium, Firefox y WebKit) con soporte multiplataforma.</p>
<br>
<p>
🧪 Tecnologías Utilizadas
Playwright
JavaScript / Node.js
Typescript
Git + GitHub
</p>
<br>
<p>
🚀 Comenzando
Requisitos previos
Asegúrate de tener instalado lo siguiente:
-Node.js (recomendado v16.x o superior)
-npm o yarn (incluido con Node.js)
</p>
<br>
<p>
Instalación
Clona el repositorio:
-git clone https://github.com/w2k31984/Playwright_practices.git 
-Ingresa al directorio:
bash> cd Playwright_practices
</p>
<br>
<p>
Instala las dependencias necesarias (si aplica):
-bash>npm install
-Ejecuta los scripts desde el directorio correspondiente:
-npx playwright install
-node nombre-del-script.js
</p>
