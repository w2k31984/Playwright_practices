🎭 Playwright Practices
Este es un repositorio dedicado al aprendizaje y práctica del uso de Playwright , una herramienta potente y moderna para automatizar navegadores web (Chromium, Firefox y WebKit) con soporte multiplataforma.

🧪 Tecnologías Utilizadas
Playwright
JavaScript / Node.js
Typescript
Git + GitHub
🚀 Comenzando
Requisitos previos
Asegúrate de tener instalado lo siguiente:

Node.js (recomendado v16.x o superior)
npm o yarn (incluido con Node.js)
Instalación
Clona el repositorio:
bash

git clone https://github.com/w2k31984/Playwright_practices.git 
Ingresa al directorio:
bash

cd Playwright_practices
Instala las dependencias necesarias (si aplica):
bash

npm install
Ejecuta los scripts desde el directorio correspondiente:
bash

npx playwright install
node nombre-del-script.js
